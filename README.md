Question 1 : What is the difference between getElementById, getElementsByClassName, and querySelector / querySelectorAll?

    getElementById - Using it we can select an element using id.
    getElementsByClassName - we can select multiple element using class.
    querySelector - Select first matching css selector.
    querySelectorAll - We can select all matching css selector.

Question 2 : How do you create and insert a new element into the DOM?

    we use document.createElement() to create and element and use appendChild() to insert it into the dom.

Question 3 : What is Event Bubbling? And how does it work?

    event bubbling means when an event happens on a child element and it also triggers the event on its parent elements.
